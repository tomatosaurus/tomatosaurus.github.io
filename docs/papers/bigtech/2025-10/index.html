25년 10월달 관심 가는 회사들에서 낸 논문. 특히 재밌게 본 회사들은 주석을 달았다.

Microsoft​

마이크로소프트 논문은 주제가 다양한 편. 그리고 논문들이 하나하나 엄청 개성있고 재밌다.

  • BitNet Distillation (10-18)
  • DEEP SELF-EVOLVING REASONING (10-21)
  • LOONGRL: REINFORCEMENT LEARNING FOR ADVANCED REASONING OVER LONG CONTEXTS (10-23)
  • Latent Sketchpad: Sketching Visual Thoughts to Elicit Multimodal Reasoning in MLLMs (10-29)
  • Magentic Marketplace: An Open-Source Environment for Studying Agentic Markets (10-31)
  • SWIREASONING: SWITCH-THINKING IN LATENT AND EXPLICIT FOR PARETO-SUPERIOR REASONING LLMS (10-07)

NVIDIA​

엔비디아 역시 주제가 꽤 다양하다. Fron-loading 논문이 개인적으로 인상깊었음.

  • VLA-0: Building State-of-the-Art VLAs with Zero Modification (10-18)

  • OmniVinci: Enhancing Architecture and Data for Omni-Modal Understanding LLM (10-21)

구글 논문은 LLM 관련 서비스 사용 경험 개선이라는 방향성이 보이는 것 같다. 검색 품질, inference, abuse 관련 논문들.

  • LLM-GUIDED HIERARCHICAL RETRIEVAL (10-18)
  • VISTA: A Test-Time Self-Improving Video Generation Agent (10-21)
  • Extracting alignment data in open models (10-22)
  • Soft Instruction De-escalation Defense (10-27)
  • ATLAS: ADAPTIVE TRANSFER SCALING LAWS FOR MULTILINGUAL PRETRAINING, FINETUNING, AND DECODING THE CURSE OF MULTILINGUALITY (10-29)
  • Supervised Reinforcement Learning: From Expert Trajectories to Step-wise Reasoning (10-31)
  • Judging with Confidence: Calibrating Autoraters to Preference Distributions (10-07)

Amazon​

아마존은 구글과 비슷하게 서비스 경험 개선 방향의 연구들이 이번에 발표되었다. 구글은 gemini를 포함한 직접적인 서비스를 고려한 것 같고 아마존은 클라우드에서 제공하는 LLM 관련 서비스가 이유가 아닐까 싶다.

  • DISTRACTOR INJECTION ATTACKS ON LARGE REASONING MODELS: CHARACTERIZATION AND DEFENSE (10-21)
