# 2025-11-24 papers review

## 1. [OpenMMReasoner](https://huggingface.co/papers/2511.16334)
### Pushing the Frontiers for Multimodal Reasoning with an Open and General Recipe

**투명한 데이터 + RL 레시피 OpenMMReasoner** 제안/공개. 

GRPO, DAPO, GSPO 비교 -> GSPO 선정.


## 2. [SAM 3](https://huggingface.co/papers/2511.16719) `meta`
### Segment Anything with Concepts

SAM 1,2 는 이용자 가원하는 단일 객체 분할에 집중함. 특정 개념에 해당하는 모든 인스턴스를 찾는 능력이 적었음.

SAM 3D와 같이 대규모 데이터 구축 프로세스를 새로 도입하여 데이터셑 구축.

모델 아키텍쳐는 DETR 기반 아키텍쳐 사용. Detector 와 Memory-based Tracker로 구성.

https://huggingface.co/facebook/sam3 : try, try!


## 3. [WorldGen](https://huggingface.co/papers/2511.16825) `meta`
### From Text to Traversable and Interactive 3D Worlds

흠 메타는 방향성이 보이는 것 같기도할 것 같다. 논문들을 쭉 이어서 보면.

3D 생성 논문인데 논문의 그림을 보면 현실보다는 메타..현실 느낌이 난달까? 

메타에서 내놓고 있는 segmentation, segmentation3d, worldgen 을 포함한 모델들이 모두 VR, AR에 적용됐을 떄 파급력이 대단할 것 같다. 내가 보고있는 화면의 물체가 갑자기 살아 움직일 수 있다!
