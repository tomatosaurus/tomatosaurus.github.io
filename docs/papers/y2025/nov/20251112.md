# 2025-11-12 papers review

## 1. [The Path Not Taken](https://huggingface.co/papers/2511.08567) `meta`
### RLVR Provably Learns Off the Principals

RLVR과 SFT와 다르게 동작하지만 모델의 성능을 오히려 크게 향상시킨다.
- RLVR은 모델 parameter를 크게 변경시키지 않는다. (sparse)
-sparsity는 표면적인 현상일 뿐 RLVR 업데이트는 모델이 선호하는 특정 파라미터 영역으로 집중된다.

테스트를 해 본 결과,
- RLVR은 SFT로 학습된 주요 영역을 의도적으로 피한다.
- SFT로 학습된 영역은 principal weights를 직접적으로 업데이트하며 동작한다.
- RLVR 은 principal weights를 피해서 업데이트. 스펙트럼 변경이 적다.
